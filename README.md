# Student Life & Study Documentation Generator VS Code Extension

> Auto-generate API docs, README files, and inline comments from Student Life & Study code patterns. Saves 2-3 hours per week on documentation.

## Features

- Core functionality

## Installation

### From VS Code Marketplace
1. Open VS Code
2. Press `Ctrl+P` (or `Cmd+P` on Mac)
3. Type `ext install digitalforge.c6885404`
4. Press Enter

### From VSIX file
1. Download the `.vsix` file
2. In VS Code: `Extensions > ··· > Install from VSIX`
3. Select the downloaded file

## Usage

Press `Ctrl+Shift+P` and search for `Student Life & Study Documentation Generator VS Code Extension` to see all available commands.

**Keyboard shortcut:** `Ctrl+Shift+Alt+A` (or `Cmd+Shift+Alt+A` on Mac)

## Settings

| Setting | Default | Description |
|---------|---------|-------------|
| `c6885404.enabled` | `true` | Enable/disable the extension |
| `c6885404.autoRun` | `false` | Auto-run on file open |
| `c6885404.logLevel` | `"info"` | Log verbosity |

## Requirements

- VS Code 1.74.0 or higher
